📊 Customer Churn Analysis

📌 Overview

This project focuses on analyzing customer churn data to identify the key factors that lead to customer attrition. The objective is to generate actionable insights that help improve customer retention and support data-driven decision-making.

---

🎯 Objectives

* Analyze customer behavior and identify churn patterns
* Perform data cleaning and exploratory data analysis (EDA)
* Build interactive dashboards for business insights
* Provide recommendations to reduce churn

---

🛠️ Tools & Technologies

* MySQL – Data storage and querying
* Python – Data cleaning and analysis (Pandas, NumPy)
* Power BI – Data visualization and dashboard creation

---

📂 Dataset Features

* Customer ID
* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Has Credit Card
* Is Active Member
* Estimated Salary
* Churn (Target Variable)

---

🔍 Analysis Performed

* Churn rate calculation
* Customer segmentation (Active vs Inactive)
* Demographic analysis (Age, Gender, Geography)
* Product usage impact on churn
* Correlation analysis of key variables

---

📊 Dashboard Highlights

* Overall Churn Rate KPI
* Customer Segmentation
* Region-wise Churn Analysis
* Interactive filters for dynamic insights

---

💡 Key Insights

* Inactive customers have a higher probability of churn
* Customers with fewer product engagements are more likely to leave
* Certain regions show higher churn trends
* Customer activity and engagement are critical for retention

---

🚀 Business Recommendations

* Improve engagement strategies for inactive users
* Provide personalized offers for high-risk customers
* Focus retention efforts on high churn regions
* Enhance overall customer experience
